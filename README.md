# t05-project

## Codes:

- Ocultation: `ocultation/ocultation.ipynb`: Reads a AVI file from Nina and detects the occultation event, then plots the light curve and saves it as a CSV file. Author: Gustavo Schwarz. https://github.com/schwarzam.

- Simple light curve extraction: `light_curve/create_lightcurve.ipynb`: Reads a folder with FITS files (already debayered) and extracts the light curve of a target star relative to comparison stars, then saves it as a CSV file. Author: Gustavo Schwarz.
